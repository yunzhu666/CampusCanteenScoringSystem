# 校园食堂评分系统

一个基于Web的校园食堂综合评分系统，采用问卷形式对食堂进行全面评估，完成后给出评分和夸张幽默的评价。

## 功能特点

### 🎯 核心功能
- **完整评分体系**：基于《学校食堂综合评分表 (百分制) - V2.0》标准
- **六大评估维度**：食品品质、性价比、多样性、环境设施、服务运营、制度包容性
- **实时评分计算**：滑块调整即时显示各部分得分
- **夸张评价系统**：根据总分生成幽默夸张的评价语
- **不合理规定扣分**：针对奇葩规定的专项扣分项

### 🎨 设计特色
- **简介优雅风格**：现代化UI设计，响应式布局
- **直观交互**：滑块评分，实时反馈
- **视觉层次**：清晰的分类和得分展示
- **打印友好**：支持打印评分结果和标准

### 🔗 多页面系统
- **主评分页面** (`index.html`)：完整的问卷评分系统
- **标准详情页** (`standard.html`)：详细的评分标准说明
- **页面间跳转**：方便的导航链接

## 评分标准体系

### 一、食品品质与安全 (30分)
- 新鲜度 (10分)
- 烹饪水平 (10分)  
- 食品安全 (10分)

### 二、价钱与性价比 (15分)
- 定价合理性 (10分)
- 分量充足性 (5分)

### 三、多样性 & 营养搭配 (15分)
- 菜品种类 (8分)
- 营养均衡 (7分)

### 四、环境与设施 (10分)
- 清洁卫生 (5分)
- 舒适度 (5分)

### 五、服务与运营 (15分)
- 供需与效率 (5分)
- 秩序管理 (5分)
- 服务态度 (5分)

### 六、制度与包容性 (15分)
- 分级性/平等性 (5分)
- 限制合理性 (5分)
- 特殊需求照顾 (5分)

### 【新增】不合理规定 (扣分项，最高扣10分)
- 强制站着吃饭 (-3分)
- 按班级分区限制就餐自由 (-4分)
- 规定极短的用餐时间 (-5分)
- 禁止使用手机等电子设备 (-3分)
- 强制购买指定套餐 (-4分)

## 评分等级

| 分数范围 | 等级 | 评价特点 |
|---------|------|----------|
| 90-100分 | 卓越食堂 | "食堂界的爱马仕！" |
| 80-89分 | 优秀食堂 | "校园里的宝藏食堂！" |
| 70-79分 | 达标食堂 | "还算过得去..." |
| 60-69分 | 待改进食堂 | "需要好好改进啊！" |
| 60分以下 | 不合格食堂 | "拉爆了！这能吃得下去饭？" |

## 使用说明

### 快速开始
1. 直接打开 `index.html` 文件即可使用
2. 无需安装任何依赖
3. 所有功能在浏览器中运行

### 评分流程
1. **填写基本信息**：输入食堂名称和评价日期
2. **调整各项评分**：使用滑块对每个项目进行评分
3. **选择扣分项**：勾选存在的不合理规定
4. **计算总分**：点击"计算总分"按钮
5. **查看结果**：系统显示总分、等级和夸张评价

### 页面导航
- 主页面 → 点击"查看详细评分标准"进入标准详情页
- 详情页 → 点击"返回评分系统"回到主页面
- 两个页面底部都有GitHub开源地址链接

### 代码亮点
```javascript
// 夸张评价生成逻辑
function updateEvaluationText(canteenName, totalScore, gradeInfo, categoryScores, penaltyScore) {
    if (totalScore >= 90) {
        return `哇塞！${canteenName}简直是食堂界的爱马仕！`;
    } else if (totalScore >= 80) {
        return `不错不错！${canteenName}是校园里的宝藏食堂！`;
    } else if (totalScore >= 70) {
        return `嗯...${canteenName}还算过得去。`;
    } else if (totalScore >= 60) {
        return `啊这...${canteenName}需要好好改进啊！`;
    } else {
        return `拉爆了！${canteenName}这能吃得下去饭？`;
    }
}
```
